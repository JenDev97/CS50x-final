<h2>Page Functionality</h2>
</div>

<p style="font-size: 16px; line-height: 1.6;">
This is a simple password generator that allows users to create secure,
random passwords using customizable options.
</p>

<hr style="border-top: 1px solid #ddd;">

### Features

<ul>
  <li>Select a password length between <strong>4 and 32 characters</strong></li>
  <li>Generate a random password based on selected criteria</li>
</ul>

<hr style="border-top: 1px dashed #ccc;">

### Password Requirements

To generate a password, users must:

<ul>
  <li>Choose a length between <strong>4 and 32</strong> characters</li>
  <li>Select at least <strong>one</strong> character type:
    <ul>
      <li>Uppercase letters</li>
      <li>Lowercase letters</li>
      <li>Numbers</li>
      <li>Symbols</li>
    </ul>
  </li>
</ul>

<p>
Users may select as many character types as they want, but
<strong>at least one option must be selected</strong>.
</p>

<hr style="border-top: 2px dotted #999;">

### Output

<p>
Once the options are chosen, the password generator will produce a
<strong>random password</strong> that matches the selected characteristics.
</p>
